Mailing capturador de email em ajax e exporta para CSV.
 
== Description ==
 
Capture emails e salve no banco de dados para depois
exportá-los em CSV.

Você pode usar o widget Formulário Mailling, ou poderá
usar a shortcode [form_mailing].

--------------------------------------------------------------------------

Capture and save emails in the database for later 
export them in CSV format. 

You can use Form Mailling widget, or you can 
use the shortcode [form_mailing].
 
== Installation ==
 
* Baixe o plugin.
* Extraia o conteúdo e coloque na pasta plugins ou
utilize a opção enviar no menu plugins.
* Ative. E pronto!
-----------------------
* Download the plugin. 
* Extract the contents and place in plugins folder or 
use the send option in the menu plugins. 
* Enable. And voila!
